 double         0.0d                 8 bytes

 Wrapper class
Helps to use primitive data types as objects

Auto boxing
primitive data type to wrapper class
int number = 5;
Integer numberWrap = number; // numberWrap is a object


Unboxing
wrapper class to primitive data
int num = numberWrap;
